Lab 1

1. Yêu cầu: Chọn 1 đối tượng bất kỳ trong thế giới thực, ví dụ như sinh viên, môn học, giảng viên,… để thiết kế trên phần mềm một chức năng trên web để thêm, sửa, xóa, tìm kiếm cho đối tượng đó.

2. Upload tất cả lên Azure / github trong menu Lab1 theo thứ tự:  Tab Yêu cầu: copy lại nội dung của phần 1 vào đây (trình bày cho đẹp)
 Tab Thiết kế: để link tới các file sau đây:
o Thiết kế trên giấy (chụp lại ảnh)
o Thiết kế trên phần mềm thiết kế
 Tab Hiện thực: chuyển tất cả thiết kế ở trên ở trên sang html

Chú ý khi làm bài tập  Sử dụng Figma để thiết kế trên toàn bộ môn học.  Phải thiết kế trên giấy nháp trước rồi thực hiện thiết kế trên phần mềm sau (cung cấp cả 2 file này trong Tab Thiết kế theo hướng dẫn trên lớp)  Đặt tên file theo định dạng lab1.fig, lab2.fig... Nếu có nhiều file thì: lab1_1.fig, lab1_2.fig.  Đặt tất cả các file thiết kế vào thư mục có tên: maSV_design (000111222_design)
